# 🤖 Document Q&A Chatbot using LangChain & RAG

## 📌 Project Overview
A Retrieval-Augmented Generation (RAG) chatbot that answers 
questions from documents using LangChain, FAISS vector database, 
and HuggingFace embeddings.

## 🛠️ Tools & Libraries
- Python
- LangChain & LangChain-Community
- FAISS (Vector Database)
- HuggingFace Embeddings (all-MiniLM-L6-v2)
- GPT-2 (Local LLM via HuggingFace)
- Google Colab

## 🔍 How It Works
1. Load and split document into chunks
2. Convert chunks to vectors using HuggingFace embeddings
3. Store vectors in FAISS database
4. On query: retrieve relevant chunks + generate answer with LLM

## 💡 Key Concepts Demonstrated
- Retrieval-Augmented Generation (RAG)
- Vector embeddings & semantic search
- FAISS vector database
- LangChain LCEL pipeline
- Local LLM inference

## 🚀 How to Run
1. Open in Google Colab
2. Run all cells in order
3. Ask questions in the final cell
